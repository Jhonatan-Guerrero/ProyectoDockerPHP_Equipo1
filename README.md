# ProyectoDockerPHP_Equipo1
Proyecto desarrollado en equipo que implementa un entorno de desarrollo utilizando Docker, integrando PHP puro (sin frameworks) y MySQL en contenedores independientes pero comunicados entre sí.


## 1. Fork del repositorio

Cada integrante del equipo debe realizar un **Fork** del repositorio original:

1. Ir al repositorio en GitHub
2. Dar clic en el botón **Fork**
3. Se creará una copia en su cuenta personal

## 2. Clonar el repositorio

En su computadora, cada integrante debe clonar su fork

## 3. Una vez descargado se tiene que ingresar a la carpeta mediante la terminal que utilizan para utilizar Docker, **SOLO LA CARPETA NO A NINGUN ARCHIVO**

## 4. Una vez dentro de la carpeta ejecutar el siguiente comando
  **docker compose up --build**

## 5. Que pasaria si payaso de rodeo en la antigua ROMA 🔢

## 6.Si la instalacion ni tuvo ningun error usar la siguiente URL:
  **http://localhost:8000/**

## 8.Crear una rama nueva, no hacer modificaciones de codigo en Main (Master)

## 7.Despues abrir el archivo con Visual Studio Code, agregar su funcionalidad utilizando los nombres de los archivos que ya existen en los botones.
  Las tablas se agregan en init.sql Agregar tablas si son necesarios.

## 8.Realizar el pull REquest al repositorio original.
